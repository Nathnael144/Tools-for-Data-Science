      Introduction to R and RStudio


R and Its Capabilities
- R is a statistical programming language used for data processing, statistical inference, and machine learning.
- It supports data import from various sources, making it a preferred choice for data scientists due to its ease of use and strong visualization capabilities.

RStudio Environment
- RStudio is an integrated development environment that enhances productivity with R, featuring a syntax-highlighting editor, console, and workspace.
- It includes tabs for managing files, plots, packages, and help resources, facilitating a smooth coding experience.

Popular R Libraries for Data Science
- Key libraries include dplyr for data manipulation, stringr for string operations, ggplot for data visualization, and caret for machine learning tasks.
- A virtual RStudio environment is provided to practice and apply learned concepts without installation hassles.
